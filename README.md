# EasyMvvm
## 基于mvvm的标准化架构工程
### 工程介绍
#### 1、app: 壳工程，用来打包编译和处理侵入性比较强的三方框架（如微信分享） 依赖home工程

#### 2、home: 业务代码主工程 依赖webview工程

#### 3、webview: 独立进程的webview封装 依赖common工程

#### 4、common:技术组件（与业务无关） + 公共
##### （1）依赖第三方框架
##### （2）存放base相关的基类
##### （3）存放公共资源
##### （4）存放公共的工具类
##### （5）存放网络模块的封装

#### 其他模块:按需自行扩展
